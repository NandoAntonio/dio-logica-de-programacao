# Desafio de Lógica - DIO (Criador de Classes do Heroi - Jogo)

Resolução do desafio de lógica de programação proposto pela Digital Innovation One (DIO). Este é o meu primeiro repositório no GitHub, focado na aplicação prática de fundamentos de desenvolvimento e Programação Orientada a Objetos (POO).

## 🚀 Sobre o Projeto
O objetivo do desafio foi estruturar uma classe genérica que represente um herói de uma aventura, alterando dinamicamente a mensagem e o tipo de ataque com base na classe selecionada (Mago, Guerreiro, Monge ou Ninja). O projeto me permitiu consolidar conceitos essenciais de arquitetura de código e controle de fluxo.

### Tecnologias e Conceitos
* **Linguagem:** JavaScript
* **Estruturas:** Variáveis, Classes, Objetos, Métodos e Estruturas de Decisão (`switch/case`).
* **Ferramentas:** Git e GitHub para controle de versão.
