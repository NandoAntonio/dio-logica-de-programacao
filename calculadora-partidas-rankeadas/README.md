# Desafio de Lógica - DIO (Calculadora de Partidas Rankeadas)

Resolução do segundo desafio de lógica de programação proposto pela Digital Innovation One (DIO). Este projeto reforça o uso de funções e manipulação de dados para estruturar o saldo de um jogador.

## 🚀 Sobre o Projeto
O objetivo do desafio foi criar uma calculadora que recebe o histórico de vitórias e derrotas de um jogador, calcula o saldo de partidas e determina a sua categoria (elo) no ranking competitivo. O projeto foca na organização e reutilização de código.

### Tecnologias e Conceitos
* **Linguagem:** JavaScript
* **Estruturas:** Funções com parâmetros, operadores aritméticos e tomada de decisão.
* **Ferramentas:** Git e GitHub para versionamento de código.
