# Desafio de Lógica - DIO (Classificador Nivel do Heroi)

Resolução do desafio de lógica de programação proposto pela Digital Innovation One (DIO). Este é o meu primeiro repositório no GitHub, focado na aplicação prática de fundamentos de desenvolvimento.

## 🚀 Sobre o Projeto
O objetivo do desafio foi estruturar a lógica para classificar o nível de um herói com base na sua quantidade de experiência (XP). O projeto me permitiu consolidar conceitos essenciais de sintaxe e controle de fluxo.

### Tecnologias e Conceitos
* **Linguagem:** JavaScript
* **Estruturas:** Variáveis, condicionais (`if/else`, `switch/case`) e funções.
* **Ferramentas:** Git e GitHub para controle de versão.
